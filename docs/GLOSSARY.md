# Glossary

- LRC: Lyrics format using `[mm:ss.xx]` timestamps.
- Enhanced LRC: word-level timing using `<mm:ss.xx>` markers.
- PoW: Proof of Work challenge used by LRCLIB anti-spam flow.
- Tap-to-sync: user marks timestamps while playback is running.
- Peaks: precomputed amplitude values used to render waveform quickly.
